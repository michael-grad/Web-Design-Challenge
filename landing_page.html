Summary: Latitude vs. X

In this project, weather attributes were analyzed with respect to proximity to the equator. More than 500 locations were pulled from OpenWeatherMap API for the source data.

After compiling the dataset, Matplotlib was used to create charts comparing temperature, humdity, cloudiness, and wind speed against latitude. This site provides the plots and source data of the analysis along with descriptions of any trends.
